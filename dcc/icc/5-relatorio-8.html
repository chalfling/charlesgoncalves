Relatório 8 - Linguagens de Programação

Palestrante: Fernando Pereira

O professor introduziu a palestra nos explicando que o termo liguagem de programação é um conjunto de regras ou instruções usadas para definir um pograma de computador. Em se tratando de uma língua, também possui vocabulário, gramática e semântica. Através das linguagens de programação é traduzido pelo compilador para uma linguagem que o computador entende, que é a linguagem de máquina, constituiada apenas por zeros e uns

A linguagem de programação surgiu para substituir de forma pratica os cartões perfurados que não apresentavam agilidade no processo. A linguagem programação é uma forma rápida, elegante, e eficiente de se traduzir para linguagem de maquinas.

Uma das primeiras linguagens a surgir foi FORTRAN (Formula Translator), que foi muito difundida dentro do mundo científico, mas que não foi um grande sucesso, pois grande parte das pessoas relutaram em aprendê-la, visto que a mesma era muito diferente de Assembly.

Por volta de 1970 surgia uma das linguagens mais utilizadas no mundo, até hoje, a linguagem C. Além de apresentar uma sintaxe muito melhor, a linguagem era mais próxima do hardware, ou seja, semelhante ao assembly e ainda próxima da língua inglesa).

Já atualmente a linguagem de programação considerada mais utilizada no mundo é a linguagem JAVA (lançada em 1995 e desde então sendo constantemente atualizada).

Existe o LLP(Laboratório de Linguagem de Programação) que é o laboratório focado ao estudo das linguagens de programação dentro da UFMG logo a matéria Linguagens de Programação é oferecida para os alunos do DCC.
